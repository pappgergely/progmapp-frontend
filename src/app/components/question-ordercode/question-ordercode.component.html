Az itt található kód egy Java osztály teljes kódja kell legyen. Az osztály kötelezően az org.progmatic.sourcequiz.classtotest package-ben kell legyen. A kód tetszőleges Java kódot tartalmazhat (de nem férhet hozzá a fájlrendszerhez, a hálózathoz, és általában nem csinálhat olyasmit, ami a Java Security manager korlátozni tud). A kódból az üres sorok és a sorvégi, soreleji whitespace karakterek törlődni fognak.

Az itt található unit teszt kód kötelezően az org.progmatic.sourcequiz.test packageben kell legyen. A kód JUnit5 unit tesztet tartalmazzon, és a fenti forráskódban definiált osztályt kell hogy letesztelje. A unit tesztben értelemszerűen példányosíthatod a fenti osztályt, és használhatsz JUnit5-ös importokat. Ha nem adsz meg Unit tesztet, akkor az ellenőrzés csak azt fogja vizsgálni, hogy a hallgató álltal beállított sorrend megegyezik-e az általad megadottal.
